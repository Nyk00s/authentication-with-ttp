Executes asymmetric verification steps and handles cross-network delivery of dynamic session keys.

Asynchronously challenges the client entity to establish valid identities. On success, wraps a fresh 32-byte pseudorandom symmetric key inside distinct RSA envelopes optimized for both participants.

Evaluates entity parameters, constructs, signs, and registers an X.509 identity token certificate.

Validates the entity name footprint by resolving asymmetric identity tokens through the local private RSA context key. Locks existing IDs to enforce unique identities.
